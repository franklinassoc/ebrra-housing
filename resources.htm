Process Overview

EBR RA Housing Rehab, Construction and Inspections

Initial Inspections

  1. EBR RA staff access the "New Request Form" page of this website to enter each new case; clicks Submit.
  2. A confirmation email is sent to staff person submitting the request and to Franklin Associates, alerting them to the request.
  3. System creates a case folder within Google Drive. 
  4. System generates a summary report of the request data and sends it to Google Drive. It is placed in the corresponding case folder.
  5. EBRRA staff access this Google Drive case folder and upload supporting documents for each property, as required.
  6. Franklin staff screen requests for completeness; requesting supplemental information as required.
  7. When complete, Franklin staff forward a notification to The Workforce Group. Case is queued for initial inspection scheduling.
  8. Inspection findings are summarized in a report. The Workforce Group delivers to Franklin Associates. It is reviewed and, if acceptable, stored in the Google Drive and a notice forwarded to EBR Redevelopment Authority.
  9. Decision is made whether case qualifies for repairs or renovation.
